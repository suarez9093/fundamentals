# What is the difference between SASS and SCSS?

- SASS (Syntactically Awesome Style-Sheets)

  - Original Language
  - Shorter syntax
  - Have to indent correctly
  - Doesn't have any curly braces or semi-colins.

- SCSS (Sassy CSS)
  - Newer syntax
  - Closer to CSS

```CSS
/* SCSS */
.container {
    float: left;
    width: 100%;
    p {
        color: #333;
    }
}

/* SASS */
.container
    float: left
    width: 100%
    p
      color: #333
```

# What is caching?

A way to remember things

Reasons for caching:

1. Save network traffic
2. Save cpu cycles
3. Save database lookup

Downsides:

1. Stale data
2. Adds complexity
3. Hard to debug

## Types of caching:

### Client side

1. browser
2. service workers / SPAs

### Network

1. DNS
2. CDN
3. HTTP

### Server

1. Object
2. Database
